Unit 1: Binary, Errors, and Images

  • Binary: relating to, using, or expressed in a system of numerical notation that has 2 rather than 10 as a base..
  • Overflow Error: a computing error that happens when a program tries to store or process a value that is larger than the maximum or too small for the minimum value that a fixed-size data type can hold, causing incorrect results, crashes, or security issues
  • Rounding Error: the difference between an exact number and its rounded or approximated value.
  • Lossless Compression: reduces file sizes by rewriting data more efficiently, allowing the original file to be perfectly reconstructed after decompression.
  • Lossy Compression: reduces file sizes by rewriting data more efficiently, allowing the original file to be perfectly reconstructed after decompression
  • Metadata: Data describing other data. 24 bits.

Unit 2: The Internet

  • Computing Device: Machine that runs programs (PC, tablet, router).
  • Computing System: Devices + programs working together.
  • Bandwidth: Max data sent per second.
  • Fault Tolerant: Works even if part breaks.
  • TCP vs UDP: TCP is reliable, UDP is faster but riskier.
  • Encryption: Protects privacy with secure codes.
